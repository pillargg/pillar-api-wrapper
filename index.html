Pillar

Base API Class. Import with: 

from pillar import Pillar

__init__(self, token=None, base_url='') special

Initialization function for the class.

The variable token is the API token needed for the API itself. The variable base_url is the url base URL for the API.

get_streamers_to_poll(self)

Gets the streamers that the application should check for. Returns a list.

get_youtube_credentials(self, twitch_username)

Gets the YouTube credentials for the user.

post_video(self, twitch_username, thumbnail_url, video_url, video_title)

Posts the video to the database. All input variables are strings and are self explanatory.
